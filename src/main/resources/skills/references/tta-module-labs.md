# Module Labs Reference

---

## Shard & Drop Labs (unlock: T4 W70)

| Lab | Effect | Max Level | Per Level |
|:----|:-------|----------:|:----------|
| Common Drop Chance | Increase the chance for bosses to drop modules | 10 | +0.10% |
| Rare Drop Chance | Increase the chance for bosses to drop rare modules | 10 | +0.10% |
| Reroll Shards | Increase the number of reroll shards that bosses drop | 100 | +1 shard |
| Daily Mission Shards | Increase the number of upgrade shards from daily mission rewards | 50 | +1 upgrade shard |
| Shatter Shards | Increase the number of shards from shattering modules (unlock: T16 W40) | 5 | +20.0% |

## Effect Ban Labs (unlock: T10 W40)

Bans allow you to exclude specific sub-stats from appearing when rerolling a module slot. Each level adds 1 ban slot for that module type.

| Lab | Module Type | Max Level | Max Bans |
|:----|:------------|----------:|---------:|
| Core Effect Ban | Core | 7 | 7 |
| Cannon Effect Ban | Cannon | 4 | 4 |
| Armor Effect Ban | Armor | 4 | 4 |
| Generator Effect Ban | Generator | 3 | 3 |

## Assist Module Labs (unlock: T19 W50)

| Lab | Effect | Max Level | Per Level |
|:----|:-------|----------:|:----------|
| Assist Module Effect | Increase the effect of the assist module's main and sub-effects | 30 | +1.0% |

Costs are in Quadrille coins (0.25Q → 7.5Q per level). As of v27.1, assist module multiplier stats are multiplicative.
