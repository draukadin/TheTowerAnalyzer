Set up Google Credentials
The Tower Analyzer uses the Google Drive and Sheets APIs to store your data. You'll need to provide an OAuth 2.0 client secret from your Google Cloud project.
How to get your credentials file
  1. Go to console.cloud.google.com
  2. Open your project → APIs & Services → Credentials
  3. Click Create Credentials → OAuth 2.0 Client ID
  4. Choose Desktop app, give it a name, click Create
  5. Click Download JSON on the credential you just created
  6. Open the downloaded file in a text editor, select all, and paste below
Configure Drive & Sheets
Enter the Google Drive folder IDs and the Google Sheets spreadsheet ID the app will use. You can find these in the URL when you open the folder or sheet in your browser.
Finding a folder ID: Open the folder in Google Drive. The URL looks like drive.google.com/drive/folders/FOLDER_ID — copy the highlighted part.
Finding a sheet ID: Open the spreadsheet. The URL looks like docs.google.com/spreadsheets/d/SHEET_ID/edit — copy the highlighted part.
Set up Claude MCP (optional)
If you use Claude Code, you can connect it directly to The Tower Analyzer. This lets Claude query your currencies, shard rates, module progress, and more — no copy-pasting needed.
What this does
  • Registers this app as an MCP server in your Claude Code config
  • Runs one command: claude mcp add tower-analyzer
  • Claude Code must already be installed for this to work
